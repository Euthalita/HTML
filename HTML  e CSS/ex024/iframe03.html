Conteúdos dentro de iframes

Dentro dos iframes é possivel colocar conteúdos, como vídeos do youtube, mapas, etc

Vídeos

Quando você puxa um vídeo do youtube, clicando em compartilhar, o iframe já é automatico

"

Maps

Também já é automatico o iframe. É possivel colocar outros mapas de outros sites, como por exemplo, o waze

Docs

É possivel puxar documentos pelo iframe, como por exemplo, do google docs
Pode ser colocado qualquer coisa, desde fotos, slides, arquivos, etc

Para carregar slides pelo docs, é necessário clicar no slide --> arquivo, compartilhar, publicar na web, conteúdo publicado e configurações
